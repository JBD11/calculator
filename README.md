#Deployment

The Site was deployed to GitHub pages

1. Go to settings tab on the repository page
2. Click on the settings tab
3. On the left hand side under Code and automation go to pages tab
4. Click on the pages tab
5. Scroll down to branch and select main and save
6. The page will refresh and generate a ribbon at the top indicating deployment and that your site is now live
7. Click the tab visit site
